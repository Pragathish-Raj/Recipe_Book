Technologies Used :
Frontend 
      ->React.js -  The core library for building the user interface.
      ->React Router -  Used for handling navigation and routing between different pages in your React app.
      ->Axios -  A library used to make HTTP requests (API calls) from your frontend to the backend server.
      ->Create React App -  Used to bootstrap and bundle the frontend project (react-scripts).
Backend
      ->Node.js - The runtime environment executing your server-side JavaScript code.
      ->Express.js - The web framework for handling API routes, HTTP requests, and server logic (usually inside your server.js and routes folder).
      ->MongoDB / Mongoose - The NoSQL database used to store data, with Mongoose acting as the Object Data Modeling (ODM) library (indicated by the models folder and initializeDB.js).
